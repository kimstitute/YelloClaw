# Final MVP Check

## Must be true

- `src/plugin-runtime.ts` is the shared orchestration source.
- `src/entrypoint.ts` is the public handler surface.
- `src/renderers/index.ts` matches the README examples.
- `src/types.ts` matches the Kakao contracts.
- `docs/final-mvp-guide.md` matches the code flow.
- `README.md` points to the correct runtime and MVP docs.

## One-user MVP

- one allowlisted user
- callback mode only
- text response available
- card response available
- quick replies available

## Completion signal

If this checklist is true and the callback flow works end-to-end, the MVP is ready.
